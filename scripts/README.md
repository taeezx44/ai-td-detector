# Data Collection Scripts

Scripts for collecting GitHub repositories with AI commit markers for the AI-TD Detector research.

## Files

- `data_collector.py` - Main script for searching and analyzing GitHub repositories
- `requirements.txt` - Python dependencies for the scripts

## Setup

```bash
cd scripts
pip install -r requirements.txt
```

## Usage

### 1. Search for repositories with AI commit markers

```bash
python data_collector.py --search "copilot" --max-repos 100 --output ai_repos.csv
```

This searches GitHub for commits containing "copilot" and saves up to 100 unique repositories to `data/ai_repos.csv`.

### 2. Analyze a specific repository

```bash
python data_collector.py --analyze "owner/repo-name"
```

This analyzes a specific repository for AI markers and saves detailed analysis to `data/analysis_owner_repo-name.json`.

### 3. Load and analyze existing data

```bash
python data_collector.py --load ai_repos.csv
```

Loads existing CSV data and analyzes a sample repository.

## Authentication

For higher rate limits, set a GitHub personal access token:

```bash
export GITHUB_TOKEN=your_token_here
python data_collector.py --search "copilot" --max-repos 500
```

Or pass it directly:

```bash
python data_collector.py --token your_token_here --search "copilot"
```

## Output

### CSV Format

The collector saves repository metadata to CSV with the following key fields:

- `name` - Repository full name (owner/repo)
- `url` - Repository URL
- `language` - Primary programming language
- `stars` - Star count
- `ai_confidence` - AI confidence level (High/Medium/Low/None)
- `ai_commit_ratio` - Ratio of AI commits to total commits
- `likely_ai_assisted` - Boolean flag if repo likely uses AI assistance
- `ai_commits_found` - Number of AI commits detected

### JSON Analysis

Detailed repository analysis includes:

- Recent commit analysis (last 100 commits)
- AI commit breakdown by confidence level
- Specific AI markers found in commit messages
- Repository metadata and statistics

## AI Detection Patterns

The script searches for these AI commit markers (case-insensitive):

- "co-authored-by: github copilot"
- "generated by copilot/chatgpt/gpt-4/claude/ai/llm"
- "ai-generated" or "ai-assisted"
- "copilot-suggested"
- "[copilot]", "[ai]", "[chatgpt]", "[claude]"
- "auto-generated by ai/llm/copilot/chatgpt"

## Rate Limits

- Unauthenticated: 60 requests per hour
- Authenticated: 5,000 requests per hour
- The script automatically handles rate limiting with delays

## Data Quality

- Repositories are filtered to 5,000–50,000 lines of code
- Must have Python, JavaScript, or TypeScript as primary language
- Must have at least 1 year of commit history
- Private repositories are excluded
